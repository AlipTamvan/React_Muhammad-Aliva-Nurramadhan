# Pembelajaran Figma

## Poin-Poin yang Dipelajari

1. **Desain Kolaboratif**  
   Figma memungkinkan kolaborasi secara real-time, sehingga saya dapat bekerja bersama tim dalam satu proyek tanpa harus bergantian mengedit. Ini mempercepat proses desain dan meningkatkan komunikasi.

2. **Prototyping yang Mudah**  
   Figma memiliki fitur prototyping yang intuitif, memungkinkan saya untuk membuat dan menguji alur interaksi dengan cepat. Saya dapat dengan mudah menambahkan animasi dan transisi untuk memberikan gambaran yang lebih jelas tentang bagaimana aplikasi atau website akan berfungsi.

3. **Kompabilitas dengan Berbagai Platform**  
   Sebagai aplikasi berbasis web, Figma dapat diakses dari berbagai perangkat dan sistem operasi tanpa perlu menginstal perangkat lunak tambahan. Ini memudahkan saya untuk bekerja di mana saja dan kapan saja.
